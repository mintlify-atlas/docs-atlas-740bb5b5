The Relio desktop application delivers native desktop performance with full offline support. Built with Tauri, it combines the power of web technologies with native system integration, providing a fast, secure, and feature-rich CRM experience.

## Technology Stack

The desktop application leverages cutting-edge technologies:

- **Tauri 2.4** - Lightweight native application framework
- **Next.js 16** - Same web app foundation
- **React 19** - Modern React features
- **Rust** - Native backend for performance and security
- **TypeScript** - Type-safe development

## Platform Support

<Tabs>
  <Tab title="Windows">
    ### System Requirements
    **Minimum:**
    - Windows 10 (64-bit) or later
    - Intel Core i3 or equivalent
    - 4GB RAM
    - 500MB available storage
    - DirectX 11 compatible graphics
    
    **Recommended:**
    - Windows 11 (64-bit)
    - Intel Core i5 or equivalent
    - 8GB+ RAM
    - 1GB available storage
    - Dedicated graphics card
    
    ### Installation
    1. Download `Relio-Setup.exe` from the downloads page
    2. Double-click the installer
    3. Follow the installation wizard
    4. Launch Relio from the Start Menu
    5. Sign in with your account
  </Tab>
  <Tab title="macOS">
    ### System Requirements
    **Minimum:**
    - macOS 11 Big Sur or later
    - Intel or Apple Silicon (M1/M2/M3)
    - 4GB RAM
    - 500MB available storage
    
    **Recommended:**
    - macOS 13 Ventura or later
    - Apple Silicon (M1 or newer)
    - 8GB+ RAM
    - 1GB available storage
    
    ### Installation
    1. Download `Relio.dmg` from the downloads page
    2. Open the DMG file
    3. Drag Relio to Applications folder
    4. Launch from Applications or Spotlight
    5. If prompted, allow the app in System Preferences > Security
    6. Sign in with your account
    
    <Note>
      The app is notarized by Apple but you may need to allow it in Security & Privacy settings on first launch.
    </Note>
  </Tab>
  <Tab title="Linux">
    ### System Requirements
    **Minimum:**
    - Ubuntu 20.04, Fedora 36, Debian 11, or equivalent
    - x86_64 or ARM64 architecture
    - 4GB RAM
    - 500MB available storage
    - GTK 3.24+
    
    **Recommended:**
    - Ubuntu 22.04 or later
    - x86_64 architecture
    - 8GB+ RAM
    - 1GB available storage
    
    ### Installation
    
    **AppImage (Universal):**
    ```bash
    # Download the AppImage
    wget https://relio.app/downloads/Relio.AppImage
    
    # Make it executable
    chmod +x Relio.AppImage
    
    # Run the application
    ./Relio.AppImage
    ```
    
    **Debian/Ubuntu (.deb):**
    ```bash
    # Download and install
    wget https://relio.app/downloads/relio_amd64.deb
    sudo dpkg -i relio_amd64.deb
    
    # Launch from applications or terminal
    relio
    ```
    
    **Fedora/RHEL (.rpm):**
    ```bash
    # Download and install
    wget https://relio.app/downloads/relio.x86_64.rpm
    sudo rpm -i relio.x86_64.rpm
    
    # Launch from applications or terminal
    relio
    ```
  </Tab>
</Tabs>

## Troubleshooting

### App Won't Start
<Tabs>
  <Tab title="Windows">
    1. Check Windows Update is current
    2. Install Visual C++ Redistributable
    3. Run as administrator
    4. Check antivirus isn't blocking
    5. Reinstall the application
  </Tab>
  <Tab title="macOS">
    1. Check System Preferences > Security & Privacy
    2. Allow the app if it's been blocked
    3. Remove from quarantine:
       ```bash
       xattr -cr /Applications/Relio.app
       ```
    4. Reinstall if issues persist
  </Tab>
  <Tab title="Linux">
    1. Check missing dependencies:
       ```bash
       ldd Relio.AppImage
       ```
    2. Install GTK and WebKit libraries
    3. Check permissions:
       ```bash
       chmod +x Relio.AppImage
       ```
    4. Run from terminal to see error messages
  </Tab>
</Tabs>

### Sync Issues
1. Check internet connection
2. Sign out and sign back in
3. Force sync from settings
4. Clear cache and re-sync
5. Contact support if issues persist

### Performance Issues
- Disable hardware acceleration if you see visual glitches
- Clear cache in settings
- Reduce sync frequency
- Update to latest version
- Check system resources (RAM, disk space)

## Uninstalling

<Tabs>
  <Tab title="Windows">
    1. Open Settings > Apps
    2. Find "Relio" in the list
    3. Click Uninstall
    4. Follow the uninstall wizard
    
    To remove all data:
    ```bash
    rmdir /s "%APPDATA%\Relio"
    ```
  </Tab>
  <Tab title="macOS">
    1. Open Applications folder
    2. Drag Relio to Trash
    3. Empty Trash
    
    To remove all data:
    ```bash
    rm -rf ~/Library/Application\ Support/Relio
    ```
  </Tab>
  <Tab title="Linux">
    **AppImage:** Simply delete the file
    
    **Debian/Ubuntu:**
    ```bash
    sudo apt remove relio
    ```
    
    **Fedora/RHEL:**
    ```bash
    sudo dnf remove relio
    ```
    
    To remove all data:
    ```bash
    rm -rf ~/.config/Relio
    ```
  </Tab>
</Tabs>
