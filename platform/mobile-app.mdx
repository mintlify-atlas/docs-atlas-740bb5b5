The Relio mobile app brings your CRM to your pocket, allowing you to manage contacts, properties, and deals from anywhere. Built with React Native and Expo, it provides a native mobile experience optimized for touch interfaces.

## Technology Stack

The mobile application leverages modern mobile technologies:

- **React Native 0.83** - Native mobile performance
- **Expo 55** - Simplified development and deployment
- **React 19** - Latest React features
- **TypeScript** - Type-safe mobile development
- **Expo Router** - File-based navigation
- **TanStack Query** - Efficient data management

## Platform Support

<Tabs>
  <Tab title="iOS">
    ### iOS Requirements
    - **iOS 15.0 or later**
    - iPhone 8 or newer
    - iPad (5th generation) or newer
    - iPod touch (7th generation) or newer

    3. Tap **Get** to download and install
    4. Open the app and sign in

  <Tab title="Android">

    - ARMv7 or ARM64 processor
    
    ### Installation
    1. Open **Google Play Store**
    2. Search for "Relio CRM"
    3. Tap **Install**
    4. Open the app and sign in
  </Tab>
</Tabs>

## Performance & Data Usage

### Battery Optimization
- Background sync is optimized for battery life
- Reduce battery usage by disabling background refresh
- Use WiFi when possible for large data syncs

### Data Usage
- Average data usage: 50-100 MB per month
- Offline mode reduces data consumption
- Images are compressed automatically
- Enable "WiFi only" mode to prevent cellular data usage

## Security

The mobile app includes enterprise-grade security:

- **End-to-end encryption** for data transmission
- **Biometric authentication** support
- **Auto-lock** after inactivity
- **Remote wipe** capability (managed by admin)
- **Secure storage** for cached data

## Troubleshooting

### App Not Syncing
1. Check internet connection
2. Pull down to refresh
3. Sign out and sign back in
4. Check app permissions

### Notifications Not Working
<Tabs>
  <Tab title="iOS">
    1. Go to **Settings > Notifications**
    2. Find **Relio CRM**
    3. Enable **Allow Notifications**
    4. Configure notification style
  </Tab>
  <Tab title="Android">
    1. Go to **Settings > Apps > Relio CRM**
    2. Tap **Notifications**
    3. Enable all notification categories
    4. Ensure battery optimization is disabled
  </Tab>
</Tabs>

### Performance Issues
- Clear app cache in settings
- Update to the latest version
- Restart your device
- Reinstall the app if problems persist
